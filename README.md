## 数据来源

- **股票行情**：`akshare`，`ak.stock_zh_a_hist()`，后复权（`qfq`），日度，2020-01-01 至今
- **市场指数**：沪深300（000300）+ 中证500（000905），来源 akshare
  - 选择中证500原因：中证500覆盖中小市值，与白酒、物流、科技类股票市值分布更接近，便于对比不同规模市场的表现
- **宏观指标**：
  - CPI同比增速（必选）：来源 akshare `ak.macro_china_cpi_yearly()`，衡量通胀压力
  - M2同比增速（自选）：来源 akshare `ak.macro_china_m2_yearly()`，货币供应量变化是股市估值的前瞻指标；M2扩张往往伴随流动性宽松，推升股价
- **财务数据**：来源 akshare `ak.stock_financial_abstract()`，ROE和资产负债率，5个年度

## 存储方式

- **基础（方式A）**：CSV 格式，适合小中规模数据，人类可读，工具兼容性最好。局限：无类型信息、大文件读写慢、不支持分区查询
- **进阶（方式B）**：Parquet 格式
  - 选择理由：列式存储在只读取部分列时性能显著优于CSV；内置Schema保留数据类型；压缩率更高。对金融时间序列按需读取列（如只读close价格）时优势明显

## 如何运行

1. 安装依赖：`pip install -r requirements.txt`
2. 运行 `01_download.ipynb` — 下载所有原始数据，生成 `download_log.txt`
3. 运行 `02_clean.ipynb` — 清洗数据，保存 CSV（方式A）和 Parquet（方式B）
4. 运行 `03_analysis.ipynb` — 描述统计、可视化、CAPM回归
5. 打开 `report.html` 阅读完整分析报告
